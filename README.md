# UPV_EARTH_PROYECTOIII

Estado actual del trabajo preparado para subir al repositorio del proyecto.

## Contenido principal

El avance desarrollado hasta ahora se concentra en [`corpus_PB/`](./corpus_PB), que contiene el corpus de referencia de Planetary Boundaries para UPV-EARTH.

## Ruta recomendada de lectura

1. [`corpus_PB/README.md`](./corpus_PB/README.md): visión general del entregable.
2. [`corpus_PB/docs/corpus_pb_methodology.pdf`](./corpus_PB/docs/corpus_pb_methodology.pdf): documentación metodológica completa.
3. [`corpus_PB/docs/pb_reference_readable_es.pdf`](./corpus_PB/docs/pb_reference_readable_es.pdf): revisión humana del corpus en español.
4. [`corpus_PB/docs/pb_reference_readable_en.pdf`](./corpus_PB/docs/pb_reference_readable_en.pdf): revisión humana del corpus en inglés.
5. [`corpus_PB/data/pb_reference.csv`](./corpus_PB/data/pb_reference.csv): recurso maestro del corpus.
