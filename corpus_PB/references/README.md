# references

Documentos fuente utilizados para construir y justificar el corpus PB.

## Contenido

- `memoria_planificación_m2.pdf`
Planificación interna del proyecto. Define la Tarea 2 y fija varios criterios metodológicos.

- `AI PBs SDGs - to resubmit.pdf`
Referencia conceptual principal para los Planetary Boundaries y sus variables de control.

- `Larosa_2025_Environ._Res._Lett._20_074032.pdf`
Referencia metodológica sobre uso de LLMs y NLP en sostenibilidad.

- `visualizing-academic-contributions-to-achieving-the-sustainable-development-goals-through-ai-the-case-of-universitat.pdf`
Referencia institucional para adaptar el corpus al perfil real de publicaciones de la UPV.

## Función dentro del entregable

Esta carpeta deja visibles las fuentes que sustentan el corpus para que cualquier persona que revise el repositorio pueda reconstruir el razonamiento metodológico sin depender de archivos dispersos fuera de `corpus_PB/`.
