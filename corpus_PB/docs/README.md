# docs

Documentación del corpus PB.

## Archivos principales

- [`corpus_pb_methodology.pdf`](./corpus_pb_methodology.pdf)
Documento metodológico principal. Es la referencia recomendada para entender el corpus con detalle.

- [`corpus_pb_methodology.tex`](./corpus_pb_methodology.tex)
Fuente en LaTeX del documento metodológico.

- [`pb_reference_readable_es.pdf`](./pb_reference_readable_es.pdf)
Versión legible en español del recurso maestro.

- [`pb_reference_readable_en.pdf`](./pb_reference_readable_en.pdf)
Versión legible en inglés del recurso maestro.

## Propósito

Esta carpeta está pensada para revisión humana y para dejar una narrativa clara del trabajo realizado. Los CSV de `../data/` son la base de trabajo; los PDFs de esta carpeta son la forma más rápida de revisar si el corpus tiene sentido.
